# Ultimate Tab Manager

A keyboard-driven Chrome Extension for tab hoarders. Stash, organize, and reopen tabs instantly — without losing anything.

## Features
- Stash tabs individually or all at once
- Organize stashed tabs into bins (folders), including nested bins
- Reopen tabs instantly, one at a time or in bulk
- Full keyboard hotkey support — no mouse required
- Customizable themes

## Tech Stack
- TypeScript, React, Tailwind, Vite
- Chrome Extension Manifest V3
- `chrome.storage.local` for local-only storage
